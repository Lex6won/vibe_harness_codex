# gg-rag — 프로젝트 AI 규칙 (골든 템플릿, Track A)

- 런타임 Python 3.12 고정. 승인 패키지만.
- 현재는 **외부 LLM 없는 로컬 텍스트 검색**만. 외부 LLM 직접 호출 코드 추가 금지.
  LLM 승격 시: 행정망은 망연계/내부 sLLM만, 프롬프트 인젝션 방어(시스템/사용자 입력 신뢰경계 분리),
  출처 표시, 입력 개인정보 마스킹, 대화 미저장 기본.
- 비밀값 `.env`만. 개인정보 평문 금지(더미).
- 외부통신·CDN 금지. 보안헤더·/health 제거 금지.
- 검증은 게이트(security-checker/gvskb).
