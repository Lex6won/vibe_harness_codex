# gg-upload — 프로젝트 AI 규칙 (골든 템플릿, Track A)

- 런타임 Python 3.12 고정. 승인 패키지만.
- 업로드 검사 레일 유지: 확장자 화이트리스트·용량 한도·안전한 파일명(경로조작 차단)·업로드 디렉터리 분리.
  업로드 파일을 정적 서빙하거나 실행 가능 위치에 저장 금지.
- 비밀값 `.env`만. 개인정보 평문 저장 금지(더미).
- 외부통신·CDN 금지. 변환은 pandas/openpyxl(승인) 로컬 처리.
- /health·보안헤더 제거 금지. 검증은 게이트(security-checker/gvskb).
