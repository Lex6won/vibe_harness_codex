# vibecode-harness

공무원 바이브코딩을 위한 스킬·에이전트·MCP 통합 하네스입니다.

## 목적
공무원이 보안, 시스템, 개발스택을 몰라도 원하는 업무를 말하면 하네스가 PRD, 설계서, DB 테이블정의서, 개발스택 문서, 보안점검, 개발 소스, 배포신청 산출물을 생성하도록 돕습니다.

## v0.1 범위
- 공무원 요구 접수
- PRD 생성
- 행정망/대민 분기
- Track A/S 중심 설계
- gg-platform-coder-lite 코딩 가드레일
- vibecode-checker 연계 지침
- 배포신청 산출물 패키지

## 시작점
`.claude/CLAUDE.md`와 `.claude/skills/gg-vibe/SKILL.md`를 사용합니다.
