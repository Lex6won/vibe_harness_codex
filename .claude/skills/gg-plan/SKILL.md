# gg-plan

## Steps
1. 한 번에 질문 하나만 한다.
2. 핵심 질문은 사용자, 목적, 주요 기능, 입력 데이터, 출력 결과, 성공 기준, 제외 범위다.
3. 답변을 요약해 되비추고 확인한다.
4. 공무원이 “빨리 진행”이라고 하면 합리적 기본값으로 채운다.
5. `01_PRD_서비스기획서.md`를 작성한다.
6. manifest의 service_exposure, data_level 후보를 갱신한다.

## Output
`_workspace/01_PRD_서비스기획서.md`
