# gg-start

## Steps
1. 사용자의 원문 요청을 업무 언어로 요약한다.
2. 첫 질문: “이 프로그램은 공무원만 쓰나요, 시민도 쓰나요?”
3. 다음 중 하나로 모드를 정한다: 상담, 설계, 개발, 검증, 신청, Lovable 전환.
4. 빠른 진행 요청 시 기본값을 제안한다.
5. `_workspace/00_input.md`와 manifest 초안을 만든다.

## Defaults
- 사용자 미정: 내부 공무원 서비스 후보
- 데이터 미정: yellow 가능성으로 보수 판정
- 외부통신 미정: 없음
