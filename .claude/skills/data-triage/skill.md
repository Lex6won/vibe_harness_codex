# data-triage

## 판정
- green: 공개자료, 샘플, 가명 테스트 데이터.
- yellow: 내부 업무자료, 개인정보 없는 이용자 입력자료.
- red: 개인정보, 민감정보, 비공개 행정정보.

## 규칙
- 답변보다 자료와 코드 증거를 우선한다.
- 대민 입력창은 개인정보 유입 가능성을 기본 가정한다.
- red면 외부 AI 입력 금지, 실데이터 개발 금지, 협의 트리거.
