# socratic-interview

## 질문 은행
1. 이 도구로 어떤 일을 편하게 하고 싶으세요?
2. 누가 쓰나요? 공무원만, 시민도, 다른 기관도?
3. 어떤 자료가 들어가나요? 이름·전화번호 같은 정보가 있나요?
4. 결과는 화면, 엑셀 다운로드, 알림 중 무엇이 필요하나요?
5. 외부 지도·문자·AI·API가 꼭 필요한가요?

## 원칙
- 한 번에 하나.
- 모르면 기본값 제안.
- 기술 용어 금지.
